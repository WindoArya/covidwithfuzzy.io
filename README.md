# covidwithfuzzy.io

en:
Implement Fuzzy logic to detect zones in Indonesian territory including red, yellow, or green zones.
By entering the name of the province in Indonesia, a fuzzy logic graph will come out with colors according to the data on the number of cases.

Id:

Dengan memasukkan nama provinsi di Indenesia makan akan keluar grafik fuzzy logic dengan warna sesuai data jumlah kasus.
